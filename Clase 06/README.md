# Animaciones por sprite sheet

- Objetivo: Animar personajes con sprite sheets y temporización.
- Ejercicio: Animación caminando (idle/walk) con cambio de frame según tiempo.

## Comentarios:

La leccion fue sencilla, pero se entendio el punto, el sistema de carga de un sprite sera de acuerdo a la resolucion 'x' y 'y' que le asignemos, ademas de tambien poder modificar el spite en tiempo real de acuerdo al evento que se configure. Aunque se uso 4 frames, se puede configurar para que vaya de una mejor manera y por ejemplo, junto al preionar una tecla, se le modifique la posicion y la animacion transcurra, y sino se queda en idle, son cosas que se pueden implementar con tiempo...
